
1. Clique no botão `Add Token`
2. Vá para a tab `Custom Token`
3. Cole o endereço do token
4. Confira se o símbolo e número de casas decimais do Token estão corretos
5. Clique no botão `Next`
6. Clique no botão `Add Tokens`
